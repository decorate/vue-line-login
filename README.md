
## npm-package-base

### Installation

With npm:

    git clone git@github.com:decorate/npm-package-base.git
    
### Replace

package.json:

    name,main,unpkg,repository,author,keywords
    peerDependencies = externalLibrary
    
### Usage

    npm install
    npm run serve
    
### Build
    
    npm run build

### Test

    npm run test
